# 👁️ راصد | Rased

> **عينك على الفرص**

منصة ذكية لتوليد إشارات تداول احترافية تلقائياً، مصممة خصيصاً للسوق السعودي.

## ✨ لماذا راصد؟
- 🎯 **دقة**: تحليل فني مدعوم بمؤشرات متعددة
- ⚡ **سرعة**: توليد ونشر الإشارات في ثوانٍ
- 🔐 **موثوقية**: شفافية كاملة مع تحذيرات مخاطر واضحة
- 🇸🇦 **محلي**: مصمم لفهم خصوصية السوق السعودي

## 🎨 هوية راصد
- اللون الأساسي: كحلي (#0F1A3C)
- لون التميز: ذهبي (#D4AF37)
- الخط العربي: تجوال (Tajawal)
- الشعار: عين + رسم بياني صاعد
