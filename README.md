# 📊 Rased AI Auto Posting

نظام أتمتة لإنشاء ونشر صور إشارات التداول بشكل احترافي.

## 🚀 المميزات

- ✅ توليد تلقائي لصور إشارات التداول
- ✅ دعم كامل للغة العربية
- ✅ تصميم احترافي جاهز للنشر
- ✅ معالجة أخطاء شاملة
- ✅ سهولة التخصيص والتعديل

## 📋 المتطلبات

- Python 3.8 أو أحدث
- Pillow (مكتبة معالجة الصور)
